# Portfólio Pessoal

Projeto desenvolvido para apresentar meu perfil como Desenvolvedor de Software, reunindo informações sobre mim, minhas habilidades técnicas, cursos realizados e projetos desenvolvidos ao longo da minha jornada na área de tecnologia.

## Objetivo

O objetivo deste projeto é servir como portfólio profissional, demonstrando minhas competências em desenvolvimento front-end e organização de aplicações modernas, além de consolidar meus estudos e experiências em um único ambiente.

## Funcionalidades

- Apresentação pessoal e resumo profissional
- Exibição das principais habilidades técnicas
- Listagem de cursos realizados e metas de aprendizado
- Apresentação de projetos front-end e back-end
- Navegação responsiva para desktop e dispositivos móveis

## Tecnologias Utilizadas

### Front-end
- React
- Next.js
- TypeScript
- Tailwind CSS

## ⚙️ Como Executar o Projeto

1. Clone o repositório:
```bash
git clone https://github.com/AdrianoATBS/portfolio.git
cd portfolio
npm install
npm run dev

```
## Acesso a Aplicação
```
http://localhost:3000
```
